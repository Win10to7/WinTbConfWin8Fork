Some WM_COMMAND messages which can be send to the taskbar.
Unused by this program, but they may be useful for other projects.
Tested on Windows 1607 with SIHost integration disabled.

eg:
SendMessageW(hWndTaskbar, WM_COMMAND, wParam, 0)

  401: Open Run dialog
  402: Log off
  403: Cascade windows
  404: Show windows stacked (tile windows horizontally)
  405: Show windows side by side (Tile windows vertically)
  415: Show desktop
  416: Do not show desktop
  419: Show desktop
  420: Run Task Manager
  421: Apply Tray properties? (seems to do nothing)
  424: Toggle lock taskbar
  430: Toggle Task View button
  436: Toggle Touch keyboard button
  506: Show shutdown dialog
  517: Lock session
41061: Apply lock taskbar setting from the registry
41094: Open Find Computers
